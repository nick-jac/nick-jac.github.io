HyperText Markup Language (HTML) is a programmimg language that controls what you see on a web page,
the HTML allows you to properly fomat a web site so that you can display links to other pages and
correctly format any text and images that are required. It is the framework that creates web pages.

What is meant by boxifying design?

Designing a web page using boxes, this how a page is constructed. Everyhting on aweb page lies
inside a box, you can have multiple boxes, you can have boxes inside boxes.

What is the box model?

The box model refers to the construction of all the boxes on a web page. Boxes are made up of four different
parts, the Content, Padding Border and Margin. The content has the html inside it, this include text and images,
the padding clears an area around the content, the border goes around the padding and the margin clears an area
outside the border. All these parts of the box allow developers to precisely create a web page for your viewing
pleasure.
