What is Grid Based design?

A grid-based design is one that works in and around the constraints of a row-and-column-based grid system to help keep your content nicely aligned.

Whats all the hype about responsive webpages?

Responsive web design (RWD) is an approach to web design which makes web pages render well on a variety of devices and window or screen sizes. Content, design and performance are necessary across all devices to ensure usability and satisfaction.

What is semantic structure?

“Semantic HTML” refers to the idea that all your HTML markup should convey the underlying meaning of your content—not its appearance.

Creating a web page with semantic structure has long been considered a best practice because it gives the web page more meaning.
